A Brief Note: Components!

See a pattern?

A-Frame is built on the concept of components:

Javascript that hooks into the component lifecycle (`init`, `tick`, `remove`, etc.).

Allows you to define more complicated behavior to make custom interactions, materials, & more!

You're probably beginning to notice a pattern here — there's a convenient built-in syntax and then the underlying A-Frame code. You can make your own contenience syntax patterns, by making your own A-Frame components! For example, we can make a color component using the following: AFRAME.registerComponent('my-color', { schema : { color : {default : '#DDAADD'} }, init : function () { console.log(this.el.setAttribute('material', 'color', this.data.color)) } });
